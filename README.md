### 〽️ Getting started

#### Frontend
```zsh
    # Clonando o repositório em sua máquina
    $ git clone https://github.com/edsonjaguiar/ignite-lab-react

    # Acessando o repositório
    $ cd ignite-lab-react

    # Instalando as dependências
    $ npm i

    # Iniciando o server
    $ npm run dev
```
